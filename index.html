Platform Features

Built for analysts.
Designed for everyone.

🌐

Fully Bilingual — AR / EN

Complete RTL Arabic and LTR English support with a single toggle. Every label, insight, and axis translates instantly.

🎨

Dark & Light Themes

Meticulously crafted color palettes for both modes — readable text, accessible borders, no invisible elements.

🎛️

Interactive Filters

Year sliders, tourist type radio buttons, and scenario selectors — every chart responds to user input in real time.

🤖

Machine Learning Embedded

K-Means segmentation and XGBoost forecasting run inside the app — no external APIs, no preprocessing step.

📊

40+ Plotly Charts

Bar, Scatter, Heatmap, Radar, Gauge, Pie, Sankey — all rendered at pixel-perfect quality with hover tooltips.

✅

Data-Validated

Every KPI, annotation, and insight verified against official SCTA and DataSaudi sources. No stale numbers.

Quick Start

Run Locally

# 1. Clone the repository $ git clone https://github.com/Goda-Emad/Saudi-Tourism-Intelligence.git $ cd Saudi-Tourism-Intelligence # 2. Install dependencies $ pip install -r requirements.txt # 3. Launch the dashboard $ streamlit run app.py

International tourism benchmarks, carbon emission coefficients, global comparisons.
